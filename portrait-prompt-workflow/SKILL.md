# Portrait Prompt Workflow

## 実行ルール

- ユーザー入力が「ポートレート」のときに開始する。
- STEP1からSTEP10まで順番に実行する。
- STEP10が完了するまで画像生成しない。
- 常に1問だけ聞く。
- 質問文には「質問」と「例（または選択肢）」だけを表示する。
- 解説・補足・余計な文章を追加しない。
- 回答受領後は次のSTEPへ進む。

## 進行テンプレート

各STEPでは次の形式で出力する。

- 質問: （そのSTEPで聞く内容）
- 例: （自由入力例 または A〜D選択肢）

## STEP1｜基本情報

質問する内容:
1. 年齢・雰囲気
2. 髪型・髪色・前髪

出力例:
- 質問: 年齢・雰囲気、髪型・髪色・前髪を入力してください。
- 例: 20代前半、清潔感のあるナチュラル系／黒髪ロング、ストレート、前髪なし

回答受領後: STEP2へ進む（画像生成しない）。

## STEP2｜顔タイプ

- 質問: 顔タイプを1つ選んでください。
- 例: A. 標準（自由カスタム） / B. 美人特化（黄金比寄せ） / C. かわいい特化（アイドル寄せ） / D. 広告CV高い顔テンプレ

回答受領後: STEP3へ進む。

## STEP3｜フェイスライン

- 質問: フェイスラインを1つ選んでください。
- 例: A. 卵型（バランス型） / B. シャープ（輪郭くっきり） / C. 丸顔（柔らかい印象）

回答受領後: STEP4へ進む。

## STEP4｜目の形

- 質問: 目の形を1つ選んでください。
- 例: A. アーモンド（自然・万能） / B. 大きめ（ぱっちり） / C. 切れ長（クール） / D. たれ目（優しい）

回答受領後: STEP5へ進む。

## STEP5｜目の印象

- 質問: 目の印象を1つ選んでください。
- 例: A. ナチュラル / B. クール / C. かわいい / D. 色っぽい

回答受領後: STEP6へ進む。

## STEP6｜鼻

- 質問: 鼻を1つ選んでください。
- 例: A. 自然で標準的 / B. やや高めで整った鼻筋 / C. 小さめで控えめ

回答受領後: STEP7へ進む。

## STEP7｜口・唇

- 質問: 口・唇を1つ選んでください。
- 例: A. 小さめで自然 / B. ややふっくら / C. 薄めでクール

回答受領後: STEP8へ進む。

## STEP8｜肌質

- 質問: 肌質を1つ選んでください。
- 例: A. ナチュラル肌（リアル重視） / B. 透明感重視 / C. ツヤ肌 / D. マット肌

回答受領後: STEP9へ進む。

## STEP9｜メイク

- 質問: メイクを1つ選んでください。
- 例: A. ナチュラル / B. 韓国風 / C. クール系 / D. しっかりメイク

回答受領後: STEP10へ進む。

## STEP10｜補足（任意）

- 質問: 補足があれば入力してください（任意）。
- 例: なし / 目元を少し明るめに

回答受領後: STEP11へ進む。

## STEP11｜日本語プロンプト生成・画像生成・確認

- STEP1〜STEP10の全回答を以下テンプレートに反映し、日本語プロンプトを生成する。
- 日本語プロンプトで画像生成する。
- 画像生成後に「この画像で問題ないか？」と確認する。
- ユーザーがOKした場合のみ英語プロンプトを出力する。
- OKが出るまで英語プロンプトを出力しない。
